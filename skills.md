# Podcast Episode Cuts Skill

## PURPOSE
Analyze a raw podcast transcript and produce a structured cut list that increases AVD without losing narrative context or conversational coherence.

## FOUNDATIONAL PRINCIPLES

### The Virality Flywheel (Never Break It)
1. Host creates psychological safety → 2. Guest reaches deep truth → 3. Truth generates gold (dense soundbites) → 4. Soundbites drive the hook for new viewers
Every cut must preserve this flywheel.

### Context Preservation Law
Every cut must pass the Context Bridge Test:
- Read 30 seconds BEFORE the cut point
- Read 30 seconds AFTER the cut point
- If a viewer jumping from before to after would be confused → DO NOT CUT

### The 4 Never-Cut Zones
1. Origin story climax
2. Number drops (specific revenue, margin, salary, growth figures)
3. Vulnerability confessions (failure, mental health, personal loss)
4. Framework reveals (named mental models or rule systems)

## THE GOLDEN EPISODE STRUCTURE

| Zone | Timestamp | Purpose | Cut Sensitivity |
|------|-----------|---------|----------------|
| Cold Hook | 0:00-1:00 | Montage of best soundbites | NEVER CUT |
| Origin & Struggle | 1:00-15:00 | Guest's lowest point | LOW — trim generic backstory |
| Turning Point | 15:00-30:00 | The moment everything changed | NEVER CUT |
| Tactical Frameworks | 30:00-60:00 | Actionable systems, rules | LOW — trim repetition |
| Contrarian Beliefs | 60:00-80:00 | Controversial takes, pushback | NEVER CUT |
| Vulnerability / Ego Death | 80:00-100:00 | Deepest personal confession | NEVER CUT |
| Future & Legacy | 100:00+ | Philosophy, vision, advice | MEDIUM |

## CUT CLASSIFICATION

### TIER 1: LINE CUTS (5-30 seconds)
- Filler removal ("um", "uh", "you know", "basically" 3+ times in 30s)
- False starts (abandoned sentences restarted with same idea)
- Redundant agreement ("yes yes", "absolutely", "100%" chains)
- Throat-clearing preambles ("That's a great question", "Let me think")
- Echo repetition (host restating exactly what guest said)
- Excessive active listening (3+ agreement markers with no follow-up)

### TIER 2: SEGMENT CUTS (30 seconds to 5 minutes)
- Tangent drift (unrelated topic for 2+ minutes with no payoff)
- Over-explained concepts (same idea 3+ ways after audience got it)
- PR mode activation (corporate tone, generic answers, no specifics)
- Promotional segments (product features without insight)
- Circular conversation (same point repeated without new angles)
- Low-energy valleys (both speakers slow down, no tension)
- Abstract/theoretical drift (hypotheticals with no examples)
- Host over-extension (personal anecdote 90+ seconds without guest engagement)

### TIER 3: STRUCTURAL CUTS
- Slow open trim (first 5 min lack hook)
- Sagging middle rescue (min 40-60 no framework/tension)
- Overstayed welcome (continues 10+ min past emotional climax)
- Redundant act merge (two segments cover similar ground)
- Cold open creation (recommend 3-4 best soundbites for montage)

## HIGH-RETENTION SPIKES (NEVER CUT)
1. The Selfish Pivot — host asks guest to solve host's personal problem live
2. The Financial Confession — sudden reveal of exact, unpolished money numbers
3. The Ego Death — guest forced to analyze deepest flaw/vulnerability

# Podcast Editorials Skill

## PURPOSE
Generate a complete editorial overlay guide — specifying WHAT to add, WHEN, and HOW.

## CORE PHILOSOPHY
- Does it help viewer understand faster? → ADD IT
- Does it make a quote more shareable? → ADD IT
- Does it reinforce a number viewer would miss? → ADD IT
- Is it just decoration? → SKIP IT
- Never deploy editorials during raw vulnerability moments. Let the camera hold.

## EDITORIAL TYPES

### TYPE 1: NUMBER POPS
Deploy every time a specific number is mentioned. Bold, large text, 2-4 seconds, quick scale-up animation.

### TYPE 2: FRAMEWORK CARDS
When host/guest introduces a mental model, rule, or acronym. Clean card overlay, 5-8 seconds.

### TYPE 3: CONTEXT BRIDGES
When conversation references something the average viewer may not know. Lower-third info bar, 3-5 seconds.

### TYPE 4: COMPARISON SPLITS
When two things are compared (X vs Y, before vs after). Split-screen graphic, 5-8 seconds.

### TYPE 5: QUOTE STAMPS
Highly quotable, contrarian, or viral-worthy statement. Full-screen or large lower-third, 3-5 seconds.

### TYPE 6: B-ROLL SUGGESTIONS
During 60+ seconds of continuous talking head. Describe footage to overlay, 5-15 seconds.

### TYPE 7: TIMELINE/JOURNEY GRAPHICS
During origin stories with specific years/milestones. Horizontal timeline, 8-12 seconds.

### TYPE 8: CHAPTER TITLE CARDS
At every major topic transition. Full-screen or partial overlay, 2-3 seconds.

## VIRAL STATEMENT DETECTION
Flag for QUOTE STAMP when matching:
- Contrarian claim (contradicts popular belief)
- Brutal truth (uncomfortable reality)
- System exposure (revealing institutional unfairness)
- Shocking money number
- Consumer psychology reveal
- Founder vulnerability
- Framework in one line
- Absolute statements ("The #1 mistake", "Top 3 red flags")

## EDITORIAL DENSITY
| Zone | Density |
|------|---------|
| Cold Hook (0:00-1:00) | VERY HIGH (4-6) |
| Origin (1:00-15:00) | MEDIUM (2-3 per 5 min) |
| Turning Point (15:00-30:00) | LOW-MEDIUM |
| Frameworks (30:00-60:00) | HIGH (4-6 per 10 min) |
| Contrarian (60:00-80:00) | MEDIUM (2-3 per 10 min) |
| Vulnerability (80:00-100:00) | VERY LOW (0-1 per 10 min) |
| Vision/Close (100:00+) | LOW (1-2 total) |

# Podcast Chapters & YouTube Skill

## PURPOSE
Generate YouTube-optimized chapters, titles, descriptions, and metadata.

## CHAPTER RULES
- Max 60 characters per title
- Lead with curiosity gap ("Why Kirana Stores Beat Amazon" not "Retail Discussion")
- Use numbers when available ("The ₹3000Cr Secret" beats "Business Model")
- Use contrarian framing
- NEVER use generic labels (ban "Introduction", "Part 1", "Conclusion", "Wrap Up")
- Each chapter title should sound like a reel hook
- 8-14 chapters per episode
- First chapter must be 0:00
- Place 3-5 seconds BEFORE the key moment
- Min chapter length: 3 minutes, max: 15 minutes

## TITLE OPTIONS (Generate 5, ranked by CTR)
| Type | Pattern |
|------|---------|
| Number Hook | "[Guest] Reveals How [Company] Makes ₹[X]Cr" |
| Contrarian | "[Guest]: [Controversial Opinion]" |
| Secret | "The [Industry] Secret Nobody Talks About" |
| Journey | "From [Struggle] to [Scale]: [Guest]'s Story" |
| Framework | "[Guest]'s [X] Rule for [Outcome]" |

Rules: Max 70 chars, include guest name, include number when possible, never clickbait the episode doesn't deliver.

## THUMBNAIL SUGGESTIONS
- Max 4-5 words text overlay
- Use most shocking number or contrarian claim
- Describe guest expression, background, text placement

## YOUTUBE DESCRIPTION STRUCTURE
1. Hook line (single biggest insight, 1 sentence)
2. Guest positioning (who + biggest credential + human element)
3. Value promise ("In this episode, you'll learn...")
4. Chapters (all timestamps)
5. Key Takeaways (5 max, contrarian framing)
6. Subscribe/social links
7. Show boilerplate
8. Hashtags (5-8 max)

First 2 lines are everything — only ~100 chars show before "Show more".

## TAGS (15-20)
Priority: Guest name → Show brand → Topic keywords → Audience interest → Long-tail phrases → Related tags
